# Description

A text-based Monopoly game modeled after the locations and features of Cornell University. Implemented in OCaml.

# Group Members
Nina Ray, May Zhou, Swathi Chakrapani, Lexi Neese

# How to compile

In order to play our game you run it in the terminal, using make and make play as the commands one after another. After you do that there should be a text based interfaces that pops up and tells you what commands you need in order to play the game.
